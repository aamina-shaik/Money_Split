# 💵Money Split

A clean and elegant web app designed for couples to track shared expenses effortlessly. Add expenses, see who paid what, and instantly know who owes whom — all in one beautiful interface.

## 📸 Preview

![App Screenshot](screeenshot.png)

## ✨ Features

- Track shared expenses between two people
- Real-time balance calculation
- Beautiful minimal UI with smooth animations
- Add, delete, and manage expenses easily
- Personalized partner names
- Fully browser-based — no backend or sign-up required
